## 📌 Overview

The **Student Performance Tracker** is an AI-powered web application that allows users to upload any student dataset and instantly perform **Exploratory Data Analysis (EDA)** and **Machine Learning predictions**.

This project transforms raw educational data into **meaningful insights and predictive intelligence**, helping identify patterns that influence student performance.

---

## 🎯 Key Objectives

* Automate **data analysis (EDA)** for any dataset
* Enable **dynamic machine learning model building**
* Predict student performance using classification/regression
* Provide **interactive visual insights**
* Build a **user-friendly analytics dashboard**

✅ Smart ML system:

## 🤖 Machine Learning Logic

* Automatically determines:

  * **Classification** → if target has limited categories
  * **Regression** → if target is continuous

* Uses:

  * RandomForestClassifier
  * RandomForestRegressor

* Evaluation Metrics:

  * Accuracy (Classification)
  * R² Score (Regression)


## 🚀 How to Run Locally

```bash id="9k2smd"
# Clone repository
git clone https://github.com/your-username/student-performance-tracker.git

# Navigate to folder
cd student-performance-tracker

# Install dependencies
pip install -r requirements.txt

# Run app
streamlit run app.py
```
