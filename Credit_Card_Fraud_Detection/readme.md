## 📌 Overview

Credit card fraud is a major challenge in the financial sector, leading to billions in losses annually.
This project presents a **Machine Learning-powered web application** that detects fraudulent transactions in real-time using an interactive and user-friendly interface.

The system not only predicts fraud but also provides **data insights, visualization, and model evaluation**, making it a complete end-to-end data science solution.

---

## 🎯 Key Objectives

* Detect fraudulent transactions with high recall
* Handle highly **imbalanced datasets** effectively
* Provide **real-time predictions** through an interactive UI
* Enable **data exploration and visualization**

